TESTNET3

Electrs testnet3

Personal Electrum server indexing the Bitcoin testnet3 chain from your sibling Bitcoin testnet3 app.

Wallet connection

Host: umbrel.local   (or your Umbrel device IP)
Port: 60031
Type: TCP (plain, no TLS)

Status

This page is static. To check sync progress, open a terminal and tail the electrs container logs:

sudo docker logs -f containers-electrs-testnet3_electrs_1

Notes

  • The matching Bitcoin testnet3 app must be installed and running at v1.2.4 or newer.
  • Initial index can take several hours; testnet3 is large.
  • RPC auth uses a shared hardcoded credential pair, so don't expose the bitcoind RPC port publicly.
