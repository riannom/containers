TESTNET4

Electrs testnet4

Personal Electrum server indexing the Bitcoin testnet4 (BIP 94) chain from your sibling Bitcoin testnet4 app.

Wallet connection

Host: umbrel.local   (or your Umbrel device IP)
Port: 60041
Type: TCP (plain, no TLS)

Status

This page is static. To check sync progress, open a terminal and tail the electrs container logs:

sudo docker logs -f containers-electrs-testnet4_electrs_1

Notes

  • The matching Bitcoin testnet4 app must be installed and running at v1.2.4 or newer.
  • Testnet4 is much smaller than testnet3, so the initial index should finish in well under an hour.
  • RPC auth uses a shared hardcoded credential pair, so don't expose the bitcoind RPC port publicly.
