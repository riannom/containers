SIGNET

Electrs signet

Personal Electrum server indexing the Bitcoin signet chain from your sibling Bitcoin signet app.

Wallet connection

Host: umbrel.local   (or your Umbrel device IP)
Port: 60051
Type: TCP (plain, no TLS)

Status

This page is static. To check sync progress, open a terminal and tail the electrs container logs:

sudo docker logs -f containers-electrs-signet_electrs_1

Notes

  • The matching Bitcoin signet app must be installed and running at v1.2.4 or newer.
  • Signet is small and produces blocks predictably, so initial index should be quick.
  • RPC auth uses a shared hardcoded credential pair, so don't expose the bitcoind RPC port publicly.
